# Learning Iteration Output (jetson_self_optimize.sh)

Run: ./scripts/jetson_self_optimize.sh --pmu --manifold

Shows:
- PMU feedback (IPC, cycles, cache_misses)
- Manifold scoring from execution feedback
- Pattern selection: 0.5 * base_score + 0.5 * pmu_score

Example flow:
  1. pmu_runner executes generated code
  2. perf stat captures metrics
  3. Feedback stored: |ipc=X|cycles=Y|cache_misses=Z
  4. Manifold scores patterns by performance
  5. select_best_pattern uses PMU-informed ranking
