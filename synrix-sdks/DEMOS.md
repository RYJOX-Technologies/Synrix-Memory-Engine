# SYNRIX Public Demos

One runnable demo per pillar. All use only the **public API** (no internals, no license/enforcement details).

| Pillar | SDK | Demo | Run |
|--------|-----|------|-----|
| **Agent memory** | agent-memory-sdk | `synrix/examples/agent_memory_demo.py` | `pip install -e .` then `python -m synrix.examples.agent_memory_demo` (set `SYNRIX_LIB_PATH`) |
| **RAG** | synrix-rag-sdk | *(coming later)* | RAG demo in progress. |
| **Robotics** | robotics-sdk | `synrix/examples/robotics_quick_demo.py` | `pip install -e .` then `python -m synrix.examples.robotics_quick_demo` (set `SYNRIX_LIB_PATH`) |
| **Robotics** | robotics-sdk | `synrix/examples/tour.py` (guided tour) | `python -m synrix` from robotics-sdk |
